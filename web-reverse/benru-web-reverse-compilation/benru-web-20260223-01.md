## 一、为什么需要爬取B站评论？

B站作为国内最大的年轻人文化社区，每天产生海量的用户评论。这些评论包含了用户对视频内容的反馈、情感倾向、热门话题等信息。对于内容创作者、市场分析师甚至学术研究者来说，评论数据都是宝贵的资源。但手动翻页复制评论显然不现实，因此编写一个自动化爬虫成为首选。

然而，B站的API接口有反爬机制，尤其是需要登录才能获取完整的评论数据（比如二级评论）。而且，为了保护账号安全，很多开发者希望每次运行程序时都重新扫码登录，而不是长期保存Cookie。本文介绍的代码正好满足了这一需求。

## 二、代码整体功能与特点

这段代码实现了一个完整的B站评论爬取流程，核心亮点包括：

  * **强制每次扫码登录** ：使用Playwright启动浏览器，用户手动扫码登录，确保账号安全且无长期Cookie泄露风险。
  * **自动处理WBI签名** ：B站API的部分接口需要WBI签名，代码内置了签名算法，无需手动计算。
  * **支持主评论和二级评论** ：可以递归获取视频下的所有一级评论，并可选地抓取每条一级评论下的所有二级回复。
  * **异步并发** ：基于  ` asyncio  ` 和  ` httpx  ` 实现异步网络请求，提高爬取效率。
  * **灵活的回调机制** ：每获取一页评论，可以通过回调函数自定义处理（如打印、保存、分析）。
  * **数据提取与保存** ：从原始API响应中提取关键字段（如评论ID、用户ID、内容、时间、点赞数等），并以JSON格式保存。

## 四、关键实现细节深入

### 1\. WBI签名的动态获取

B站的WBI密钥并不是固定不变的，它会定期更换。代码中通过两种方式获取：

  * 先尝试从浏览器的localStorage读取  ` wbi_img_urls  ` 或  ` wbi_img_url  ` /  ` wbi_sub_url  ` 。
  * 如果本地没有，则调用  ` /x/web-interface/nav  ` 接口，从返回的  ` wbi_img  ` 字段中提取。

这样就保证了每次运行时都能拿到最新的密钥。

### 2\. 强制扫码登录的实现

为什么强调“每次扫码登录”？因为有些爬虫会保存Cookie长期使用，一旦Cookie失效或账号被限制，就会影响爬取。这里使用Playwright启动浏览器，不加载任何已有的用户数据，每次都是全新上下文，用户手动扫码登录。登录成功后，Cookie只在本次运行中使用，程序退出后不保存。这既安全又灵活。

### 3\. 递归翻页与游标

B站的评论接口使用游标（  ` next  ` ）进行翻页，而不是传统的页码。第一页传入  ` next=0  ` ，返回的  ` cursor  `
中会包含  ` next  ` 字段供下一页使用，以及  ` is_end  ` 表示是否最后一页。代码中通过  ` while not is_end  `
循环获取所有页面。

### 4\. 二级评论的获取

每个一级评论可能有成百上千条二级回复，如果全部抓取会非常耗时。代码中通过  ` rcount  `
字段判断是否有回复，有的话再调用二级评论接口。二级评论接口使用页码（  ` pn  ` ）翻页，每页最多20条。

### 5\. 异步与并发控制

虽然使用了  ` asyncio  ` ，但为了不给服务器造成过大压力，代码在每页请求之间加入了  `
asyncio.sleep(crawl_interval)  ` ，默认1秒。你可以根据实际情况调整间隔。

## 五、注意事项

  * **登录超时** ：如果扫码后长时间没有检测到登录成功，脚本会超时退出。可以适当增加  ` timeout  ` 参数。
  * **请求频率** ：频繁请求可能导致IP被临时封禁，建议适当调大  ` crawl_interval  ` 。
  * **二级评论数量** ：如果视频很火爆，二级评论可能非常庞大，抓取所有会花费很长时间。可以通过修改  ` get_video_all_comments  ` 的逻辑限制二级评论页数。
  * **数据格式** ：提取的信息仅包含基本字段，如需更多信息（如用户昵称、IP属地等），可以在  ` extract_comment_info  ` 中补充。
  * **接口变更** ：B站API可能会更新，如果发现爬虫失效，可能需要调整接口地址或签名算法。

    # -*- coding: utf-8 -*-
    """
    Bilibili评论爬取核心模块（每次扫码登录版）
    功能：
    - 每次运行强制扫码登录（不保存登录状态）
    - 爬取评论后自动提取主要信息并保存为 JSON 文件
    """
    import asyncio
    import json
    import os
    import time
    import urllib.parse
    from hashlib import md5
    from enum import Enum
    from typing import Dict, List, Optional, Callable, Tuple, Any
    import httpx
    from playwright.async_api import Page, async_playwright

    # -------------------- 异常定义 --------------------
    class DataFetchError(Exception):
        """数据获取错误"""
        pass

    # -------------------- 字段枚举 --------------------
    class CommentOrderType(Enum):
        """评论排序方式"""
        DEFAULT = 0      # 仅按热度
        MIXED = 1         # 热度+时间
        TIME = 2          # 仅按时间

    # -------------------- WBI签名工具 --------------------
    class BilibiliSign:
        """B站WBI签名实现"""
        def __init__(self, img_key: str, sub_key: str):
            self.img_key = img_key
            self.sub_key = sub_key
            self.map_table = [
                46, 47, 18, 2, 53, 8, 23, 32, 15, 50, 10, 31, 58, 3, 45, 35, 27, 43, 5, 49,
                33, 9, 42, 19, 29, 28, 14, 39, 12, 38, 41, 13, 37, 48, 7, 16, 24, 55, 40,
                61, 26, 17, 0, 1, 60, 51, 30, 4, 22, 25, 54, 21, 56, 59, 6, 63, 57, 62, 11,
                36, 20, 34, 44, 52
            ]

        def get_salt(self) -> str:
            """获取混合密钥的前32位"""
            mixin_key = self.img_key + self.sub_key
            return ''.join(mixin_key[mt] for mt in self.map_table)[:32]

        def sign(self, req_data: Dict) -> Dict:
            """
            对请求参数进行WBI签名，添加wts和w_rid字段
            :param req_data: 原始参数字典
            :return: 添加签名后的参数字典
            """
            current_ts = int(time.time())
            req_data.update({"wts": current_ts})
            # 按键排序
            req_data = dict(sorted(req_data.items()))
            # 过滤值中的特殊字符 "!'()*"
            req_data = {
                k: ''.join(filter(lambda ch: ch not in "!'()*", str(v)))
                for k, v in req_data.items()
            }
            query = urllib.parse.urlencode(req_data)
            salt = self.get_salt()
            w_rid = md5((query + salt).encode()).hexdigest()
            req_data['w_rid'] = w_rid
            return req_data

    # -------------------- Bilibili API客户端 --------------------
    class BilibiliClient:
        """Bilibili API客户端，封装评论相关接口"""
        def __init__(
            self,
            headers: Dict[str, str],
            playwright_page: Page,
            proxy: Optional[str] = None,
            timeout: int = 30
        ):
            self.headers = headers
            self.playwright_page = playwright_page
            self.proxy = proxy
            self.timeout = timeout
            self._host = "https://api.bilibili.com"

        async def _request(self, method: str, url: str, **kwargs) -> Dict:
            """发送HTTP请求，自动处理JSON响应和错误码"""
            async with httpx.AsyncClient(proxy=self.proxy) as client:
                resp = await client.request(method, url, timeout=self.timeout, **kwargs)
            try:
                data = resp.json()
            except json.JSONDecodeError:
                raise DataFetchError(f"JSON解析失败: {resp.text[:200]}")
            if data.get("code") != 0:
                raise DataFetchError(f"API错误: {data.get('message')} (code={data.get('code')})")
            return data.get("data", {})

        async def get_wbi_keys(self) -> Tuple[str, str]:
            """
            从localStorage或接口获取最新的WBI密钥对
            :return: (img_key, sub_key)
            """
            # 尝试从localStorage获取
            local_storage = await self.playwright_page.evaluate("() => window.localStorage")
            wbi_img_urls = local_storage.get("wbi_img_urls", "")
            if not wbi_img_urls:
                img_url = local_storage.get("wbi_img_url")
                sub_url = local_storage.get("wbi_sub_url")
                if img_url and sub_url:
                    wbi_img_urls = f"{img_url}-{sub_url}"
            if wbi_img_urls and "-" in wbi_img_urls:
                img_url, sub_url = wbi_img_urls.split("-")
            else:
                # 回退到接口获取
                resp = await self._request("GET", self._host + "/x/web-interface/nav")
                img_url = resp['wbi_img']['img_url']
                sub_url = resp['wbi_img']['sub_url']

            img_key = img_url.rsplit('/', 1)[1].split('.')[0]
            sub_key = sub_url.rsplit('/', 1)[1].split('.')[0]
            return img_key, sub_key

        async def pre_request_data(self, params: Dict) -> Dict:
            """
            对请求参数进行WBI签名预处理
            """
            img_key, sub_key = await self.get_wbi_keys()
            return BilibiliSign(img_key, sub_key).sign(params)

        async def get(self, uri: str, params: Dict = None, sign: bool = True) -> Dict:
            """GET请求封装，自动处理签名和URL拼接"""
            if params is None:
                params = {}
            if sign:
                params = await self.pre_request_data(params)
            url = f"{self._host}{uri}"
            if params:
                url += "?" + urllib.parse.urlencode(params)
            return await self._request("GET", url, headers=self.headers)

        # -------------------- 评论相关接口 --------------------
        async def get_video_comments(
            self,
            video_id: str,
            order_mode: CommentOrderType = CommentOrderType.DEFAULT,
            next: int = 0
        ) -> Dict:
            """
            获取视频的一页主评论（新接口 /x/v2/reply/wbi/main）
            :param video_id: 视频AV号（字符串形式）
            :param order_mode: 排序方式
            :param next: 翻页游标，第一页为0，后续从返回的cursor.next获取
            :return: API返回的data部分
            """
            uri = "/x/v2/reply/wbi/main"
            params = {
                "oid": video_id,
                "mode": order_mode.value,
                "type": 1,
                "ps": 20,          # 每页数量，固定20
                "next": next
            }
            return await self.get(uri, params, sign=True)

        async def get_video_level_two_comments(
            self,
            video_id: str,
            root_comment_id: int,
            pn: int = 1,
            ps: int = 20,
            order_mode: CommentOrderType = CommentOrderType.DEFAULT
        ) -> Dict:
            """
            获取一级评论下的二级评论（子回复）
            :param video_id: 视频AV号
            :param root_comment_id: 一级评论ID (rpid)
            :param pn: 页码
            :param ps: 每页数量
            :param order_mode: 排序方式
            :return: API返回的data部分
            """
            uri = "/x/v2/reply/reply"
            params = {
                "oid": video_id,
                "type": 1,
                "root": root_comment_id,
                "pn": pn,
                "ps": ps,
                "mode": order_mode.value
            }
            return await self.get(uri, params, sign=True)

        async def get_video_all_comments(
            self,
            video_id: str,
            max_count: int = 1000,
            crawl_interval: float = 1.0,
            fetch_sub_comments: bool = False,
            comment_callback: Optional[Callable] = None,
            sub_comment_callback: Optional[Callable] = None
        ) -> List[Dict]:
            """
            递归获取视频的所有主评论（可包含子评论）
            :param video_id: 视频AV号
            :param max_count: 最大获取评论总数
            :param crawl_interval: 每页间隔（秒）
            :param fetch_sub_comments: 是否同时爬取二级评论
            :param comment_callback: 每获取一页主评论的回调，参数为(video_id, comment_list)
            :param sub_comment_callback: 每获取一页二级评论的回调，参数为(video_id, sub_comment_list)
            :return: 所有主评论列表（如果fetch_sub_comments为True，则主评论的replies字段不会自动填充子评论，但可通过回调收集所有评论）
            """
            all_comments = []
            next_page = 0
            is_end = False

            while not is_end and len(all_comments) < max_count:
                try:
                    data = await self.get_video_comments(video_id, next=next_page)
                except DataFetchError as e:
                    print(f"获取评论失败: {e}")
                    break

                cursor = data.get("cursor", {})
                is_end = cursor.get("is_end", True)
                next_page = cursor.get("next", 0)

                replies = data.get("replies", [])
                if not replies:
                    break

                # 限制数量
                if len(all_comments) + len(replies) > max_count:
                    replies = replies[:max_count - len(all_comments)]

                # 回调处理（主评论）
                if comment_callback:
                    await comment_callback(video_id, replies)

                # 如果需要子评论，为每个一级评论获取子评论
                if fetch_sub_comments:
                    for comment in replies:
                        if comment.get("rcount", 0) > 0:
                            await self.get_video_all_level_two_comments(
                                video_id=video_id,
                                root_comment_id=comment["rpid"],
                                order_mode=CommentOrderType.DEFAULT,
                                crawl_interval=crawl_interval,
                                callback=sub_comment_callback
                            )

                all_comments.extend(replies)
                await asyncio.sleep(crawl_interval)

            return all_comments

        async def get_video_all_level_two_comments(
            self,
            video_id: str,
            root_comment_id: int,
            order_mode: CommentOrderType = CommentOrderType.DEFAULT,
            ps: int = 20,
            crawl_interval: float = 1.0,
            callback: Optional[Callable] = None
        ) -> List[Dict]:
            """
            获取指定一级评论下的所有二级评论
            :param video_id: 视频AV号
            :param root_comment_id: 一级评论ID
            :param order_mode: 排序方式
            :param ps: 每页数量
            :param crawl_interval: 翻页间隔
            :param callback: 每页回调
            :return: 所有二级评论列表
            """
            all_replies = []
            pn = 1
            while True:
                data = await self.get_video_level_two_comments(
                    video_id=video_id,
                    root_comment_id=root_comment_id,
                    pn=pn,
                    ps=ps,
                    order_mode=order_mode
                )
                replies = data.get("replies", [])
                if callback:
                    await callback(video_id, replies)
                all_replies.extend(replies)
                # 判断是否最后一页
                page_info = data.get("page", {})
                total = page_info.get("count", 0)
                if pn * ps >= total:
                    break
                pn += 1
                await asyncio.sleep(crawl_interval)
            return all_replies

    async def wait_for_login(page: Page, timeout: int = 120) -> bool:
        """
        等待用户在页面上完成扫码登录
        :param page: Playwright 页面对象
        :param timeout: 最大等待秒数（默认120秒）
        :return: 是否成功登录
        """
        print("请在打开的浏览器中扫码登录 Bilibili ...")
        start_time = time.time()
        while time.time() - start_time < timeout:
            # 检查 Cookie 中是否存在登录标志 SESSDATA
            cookies = await page.context.cookies()
            for cookie in cookies:
                if cookie['name'] == 'SESSDATA' and cookie['value']:
                    print("检测到登录成功！")
                    return True
            await asyncio.sleep(2)  # 每2秒检查一次
        print("登录超时，请重试。")
        return False

    def extract_comment_info(comment: Dict[str, Any]) -> Dict[str, Any]:
        """
        从评论原始数据中提取主要信息
        :param comment: 评论原始字典（来自API）
        :return: 包含关键字段的新字典
        """
        return {
            "rpid": comment.get("rpid"),
            "oid": comment.get("oid"),
            "mid": comment.get("mid"),
            "parent": comment.get("parent", 0),      # 父评论rpid，0表示一级评论
            "root": comment.get("root", 0),           # 根评论rpid
            "ctime": comment.get("ctime"),
            "like": comment.get("like"),
            "message": comment.get("content", {}).get("message", ""),
            "replies_count": comment.get("rcount", 0)  # 该评论下的子评论数量
        }

    async def main():
        COMMENTS_FILE = "bilibili_comments.json"# 保存爬取的评论数据

        async with async_playwright() as p:
            # 启动浏览器（使用系统 Chrome，方便复用已登录状态）
            browser = await p.chromium.launch(headless=False, channel="chrome")
            # 每次运行都创建全新的上下文，不加载任何存储的登录状态
            context = await browser.new_context()
            page = await context.new_page()
            await page.goto("https://www.bilibili.com")

            # ---------- 强制扫码登录 ----------
            if not await wait_for_login(page):
                print("登录失败，程序退出。")
                await browser.close()
                return
            # --------------------------------------

            # 获取当前 cookies 并构建请求头
            cookies = await context.cookies()
            cookie_str = "; ".join([f"{c['name']}={c['value']}" for c in cookies])
            headers = {
                "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36",
                "Cookie": cookie_str,
                "Referer": "https://www.bilibili.com",
                "Origin": "https://www.bilibili.com"
            }

            # 创建 API 客户端
            client = BilibiliClient(headers=headers, playwright_page=page)

            # 目标视频 AV 号（请替换为你的实际 AV 号）
            video_aid = "116091825160447"

            # 用于收集所有评论的列表（包括一级和二级）
            all_comments_data = []

            # 定义回调函数：每获取一页评论，提取信息并存入列表
            async def on_comments(video_id, comments):
                print(f"视频 {video_id} 获取到 {len(comments)} 条评论")
                for c in comments:
                    info = extract_comment_info(c)
                    all_comments_data.append(info)
                    # 简单打印内容（可选）
                    print(f"  - {info['message'][:30]}..." if len(info['message']) > 30 else f"  - {info['message']}")

            # 获取所有评论（包含子评论）
            print(f"开始爬取视频 {video_aid} 的评论...")
            comments = await client.get_video_all_comments(
                video_id=video_aid,
                max_count=500,               # 可根据需要调整
                fetch_sub_comments=True,
                comment_callback=on_comments,
                sub_comment_callback=on_comments  # 子评论也使用相同的回调
            )
            print(f"爬取完成，共获取到 {len(comments)} 条主评论，{len(all_comments_data)} 条总评论（含子评论）")

            # 将收集到的所有评论保存到文件
            with open(COMMENTS_FILE, "w", encoding="utf-8") as f:
                json.dump(all_comments_data, f, ensure_ascii=False, indent=2)
            print(f"评论数据已保存至 {COMMENTS_FILE}")

            await browser.close()

    if __name__ == "__main__":
        asyncio.run(main())
