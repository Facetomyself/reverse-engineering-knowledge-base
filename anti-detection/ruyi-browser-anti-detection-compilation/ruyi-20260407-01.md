##  安装


    pip install ruyiPage

安装后建议先确认：


    python -c "import ruyipage; print(ruyipage.__version__)"


##  基础配置

###  1\. 最简单启动


    from ruyipage import FirefoxPage

    page = FirefoxPage()
    page.get("https://www.example.com")
    print(page.title)
    page.quit()
